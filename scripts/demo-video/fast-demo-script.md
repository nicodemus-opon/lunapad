# Fast Demo Script

Target length: 3-5 minutes. Actual generated length: about 3 minutes.

1. Open on the Sales Analytics Demo and the dependency graph.
2. Run all cells and show the notebook populating.
3. Lead with `growth_analysis`, a standard SQL cell referencing `monthly_revenue`.
4. Live-edit the SQL and rerun the cell.
5. Toggle `orders` through Stats, Table, and Chart views.
6. Show `region_performance` as the optional visual/PRQL power-up.
7. Change the dashboard Region filter and show the linked SQL detail table.
8. End in Report view.

Key framing: SQL-first, PRQL/GUI as optional power-ups, no passive scrolling.
